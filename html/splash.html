Welcome to Jagadish Robot Shop

Here you will find all of Jagadish's friends. Have a browse around and see who is here.

This is a simple example of a microservices ecommerce application. It has been built using various technologies:

  • AngularJS (1.x)
  • Nginx
  • NodeJS
  • Java
  • Python
  • Golang
  • PHP (Apache)
  • MongoDB
  • Redis
  • MySQL

When deployed into an environment monitored by Instana, these technology stacks will be automatically detected and monitored, all with minimal configuration. Every request will be traced end to end. Jagadish will keep an eye on all those metrics, events, and traces and let you know what needs your attention.

To find out more, visit the Instana site.

All the code is available on Github.
